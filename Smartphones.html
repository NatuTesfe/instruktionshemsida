Redigera på Skärmen: Mobiltelefon(Iphone)

Lär dig grunderna i att använda pekskärmen på smartphones och tablets:

  1. Tryck lätt på skärmen för att välja objekt eller öppna appar.
  2. Håll och dra för att flytta eller navigera.
  3. Använd nypa och sprida fingrarna för att zooma in och ut.
  4. Dubbeltryck för att zooma in snabbt på ett område.
  5. Svep för att bläddra genom sidor eller mellan appar.

Använda Appar

Så här installerar och använder du appar på dina enheter:

  1. Öppna appbutiken på din enhet (t.ex. App Store, Google Play).
  2. Sök efter den app du vill installera.
  3. Klicka på "Installera" eller "Hämta" för att ladda ner appen.
  4. Öppna appen genom att trycka på dess ikon på startskärmen.
  5. Följ instruktionerna i appen för att använda den.
